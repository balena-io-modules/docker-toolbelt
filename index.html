docker-toolbelt

Docker Toolbelt

A belt full of tools for working with Docker.

Docker Toolbelt extends dockerode with additional utilities to make working with Docker a breeze. This module also supports balenaEngine, notably it's delta generation feature.

Installation

Install docker-toolbelt by running:

$ npm install --save docker-toolbelt

Usage

 
import { DockerToolbelt } from 'docker-toolbelt'

const belt = new DockerToolbelt({
  host: '192.168.1.10',
  port: process.env.DOCKER_PORT || 2375,
    ...
});

const containers = await belt.listContainers()
